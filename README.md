# AI Intelligence Layer

One backend that connects Sentinel, DocuZen, and StackScope.

Ask it anything in plain English it figures out which tool to call and returns the answer.

**Status:** In progress

## What it does

Routes natural language queries to the right tool automatically.

- Sentinel — "are my services up?"
- DocuZen — "what does this document say?"
- StackScope — "what skills are trending for backend developers?"

## Stack

FastAPI · OpenAI · Python
